# etl_project
A simple Extract, Transform and Load project made with Python.

## Descripción
Este proyecto extrae datos de Museos, Salas de Cine y Bibliotecas Populares, realiza transformaciones de limpieza con Pandas y carga los resultados en una base de datos.

## Requisitos
- Python 3.x
- Ver `requirements.txt` para librerías.

## Instalación
1. Clonar el repo.
2. Crear entorno virtual: `python -m venv venv`
3. Instalar dependencias: `pip install -r requirements.txt`
